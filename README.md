## Features

## Theme

- Deep black (#0A0A0A) to dark gray gradient background

- Emerald green (#10B981) as primary accent color
- Cormorant Garamond serif for display typography
- Inter sans-serif for body text
- Sophisticated spacing and hierarchy
- Subtle staggered fade-in animations on load
